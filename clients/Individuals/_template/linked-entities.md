# Linked Entities

Status: template.

Use this file to map records that attach directly to an individual.

## Entity Routing

| Entity type | Local file | Routing rule |
| --- | --- | --- |
| Bank accounts | `bank-accounts.md` | Individual bank account records, balances, statements, account restrictions, garnishments, and payment-source context. |
| Tax filings | `tax-filings.md` | Individual tax filing records and filing status. |
| Assets | `assets.md` | Individual-owned assets, crypto, shareholdings, personal liquidity, and supporting evidence. |
| Correspondence | `correspondence.md` | Notion Correspondence records and source-system correspondence linked to the individual. |
| Expenses | `expenses.md` | Expenses, reimbursements, personal-service costs, or expense evidence attached to the individual. |
