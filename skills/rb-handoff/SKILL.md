# RB Handoff

Use this before ending a substantial RB repo session.

## Checklist

1. Update `memory/current-state.md`.
2. Update `memory/tasks.md`.
3. Update `memory/open-questions.md`.
4. Update `sources/import-log.md`.
5. Update client `open-questions.md` files if client details were touched.
6. Run `git diff --check`.
7. Note whether Notion/Drive backup was completed or blocked.
